ÉTAPE 2 — CORRECTION PORTRAIT UNIQUEMENT
-----------------------------------------
Après l'audit, corriger UNIQUEMENT le CSS du portrait.

OBJECTIF :
- L'image HeliCard.png (1920x1080, paysage, fond transparent) 
  doit s'afficher ENTIÈREMENT dans la zone illustration
- Pas de coupure, pas de déformation
- Centrée sur le personnage (pas sur le vide transparent à gauche)
- Responsive : identique sur desktop et mobile

RÈGLE CSS FINALE à appliquer dans le fichier CSS du composant carte :

.layer-character {
  position: absolute;
  top: 30px;
  left: 6px;
  right: 6px;
  bottom: 36%;
  overflow: hidden;
  background: transparent;
}

.layer-character img {
  width: 100%;
  height: 100%;
  object-fit: contain;
  object-position: center center;
  transform: scale(1.15) translateX(8%);
  transform-origin: center center;
  filter: drop-shadow(0 0 20px rgba(200, 100, 0, 0.4));
}

EXPLICATION :
- object-fit: contain   → image entière visible, pas de coupure
- object-position: center center → centrée dans la zone
- transform: scale(1.15) translateX(8%) → légère mise à l'échelle
  pour remplir la zone ET décale de 8% vers la droite pour compenser
  le vide transparent à gauche de l'image
- bottom: 36% → la zone s'arrête avant le texte du bas
