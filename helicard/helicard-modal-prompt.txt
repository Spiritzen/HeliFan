CONTEXTE
--------
La carte fantasy HeliFan existe et tourne sur localhost:5173.
Le problème : la carte est trop petite, le texte est illisible.

SOLUTION : Au clic sur la carte, une modale s'ouvre.
Elle affiche une version agrandie (80% écran) de la carte
avec tout le contenu lisible, les effets 3D actifs,
et une zone de texte détaillée à côté.


COMPORTEMENT SOUHAITÉ
---------------------
1. La carte principale sur la page reste à sa taille actuelle
   (petite, mystérieuse, avec le tilt 3D au hover)

2. Au CLIC sur la carte → modale plein écran s'ouvre
   - Fond : overlay sombre rgba(0,0,0,0.88) avec blur backdrop-filter: blur(8px)
   - Animation d'ouverture : scale(0.85) → scale(1) + opacity 0→1, durée 0.35s ease-out
   - Touche Échap ou clic sur le fond → ferme la modale

3. Dans la modale : disposition en deux colonnes
   Colonne gauche (45%) : la carte agrandie avec tilt 3D actif
   Colonne droite (55%) : panneau de détails scrollable

import { useEffect, useRef } from 'react';
import type { FantasyCard } from './types';

interface CardModalProps {
  card: FantasyCard;
  isOpen: boolean;
  onClose: () => void;
}

export function CardModal({ card, isOpen, onClose }: CardModalProps) {
  // Fermer avec Échap
  useEffect(() => {
    const handleKey = (e: KeyboardEvent) => {
      if (e.key === 'Escape') onClose();
    };
    if (isOpen) document.addEventListener('keydown', handleKey);
    return () => document.removeEventListener('keydown', handleKey);
  }, [isOpen, onClose]);

  // Bloquer le scroll du body quand la modale est ouverte
  useEffect(() => {
    document.body.style.overflow = isOpen ? 'hidden' : '';
    return () => { document.body.style.overflow = ''; };
  }, [isOpen]);

  if (!isOpen) return null;

  return (

      className="modal-overlay"
      onClick={onClose}
      role="dialog"
      aria-modal="true"

        </button>

MODIFICATION DE InteractiveFantasyCard.tsx
------------------------------------------
Ajouter en haut du composant :

  const [isModalOpen, setIsModalOpen] = useState(false);

Entourer la carte d'un bouton ou ajouter onClick sur le wrapper :

  <div
    className="card-wrapper"
    ref={wrapperRef}
    onClick={() => setIsModalOpen(true)}
    role="button"
    tabIndex={0}
    aria-label="Cliquer pour voir les détails de la carte HeliFan"
    onKeyDown={e => { if (e.key === 'Enter') setIsModalOpen(true); }}
  >

Ajouter à la fin du JSX, avant la fermeture du composant :

  <CardModal
    card={card}
    isOpen={isModalOpen}
    onClose={() => setIsModalOpen(false)}
  />

Retirer ou déplacer les boutons "Révéler le choix" et "Retourner"
dans la modale (ils ne sont plus nécessaires sur la petite carte).
Garder uniquement un texte hint discret sous la carte :
  <p className="card-hint">Cliquer pour explorer</p>

Style du hint :
  .card-hint {
    font-family: 'Cinzel', serif;
    font-size: 9px;
    color: #5A3A0A;
    letter-spacing: 2px;
    text-transform: uppercase;
    margin-top: 12px;
    animation: hintPulse 2.5s ease-in-out infinite;
  }
  @keyframes hintPulse {
    0%, 100% { opacity: 0.4; }
    50%       { opacity: 0.9; }
  }
