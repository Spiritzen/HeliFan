CORRECTION 1 — image dans la carte (InteractiveFantasyCard.css ou InteractiveCard.css)
---------------------------------------------------------------------------------------
Trouver le sélecteur qui cible l'image du portrait.
Le remplacer EXACTEMENT par :

.layer-character {
  position: absolute;
  inset: 0;
  overflow: hidden;
  display: flex;
  align-items: flex-start;
  justify-content: center;
}

.layer-character img {
  width: 100%;
  height: 100%;
  object-fit: cover;
  object-position: 65% 20%;
  filter: drop-shadow(0 0 20px rgba(200, 100, 0, 0.4));
}

EXPLICATION object-position: 65% 20%
  65% horizontal → décale vers la droite pour centrer sur le personnage
                   (évite le vide transparent à gauche)
  20% vertical   → cadre sur le haut du personnage (visage visible)
                   sans couper les cartes en bas
